Connect Supabase
Add your Supabase credentials below to enable live data sync across all agents.
Step 1 — Create your Supabase project
Go to supabase.com, create a new project, then copy your URL and anon key from Settings → API.
Step 2 — Run this SQL in your Supabase SQL editor
-- Agents table CREATE TABLE agents ( id TEXT PRIMARY KEY, name TEXT, role TEXT, tier TEXT, manager_id TEXT, goal_target INT, created_at TIMESTAMPTZ DEFAULT NOW() ); -- Policies table CREATE TABLE policies ( id TEXT PRIMARY KEY, agent_id TEXT, client TEXT, carrier TEXT, product TEXT, annual_premium INT, status TEXT, submitted_date DATE, issued_date DATE, created_at TIMESTAMPTZ DEFAULT NOW() ); -- Requirements table CREATE TABLE requirements ( id TEXT PRIMARY KEY, policy_id TEXT, agent_id TEXT, title TEXT, description TEXT, due_date DATE, resolved BOOLEAN DEFAULT FALSE ); -- Commissions table CREATE TABLE commissions ( id TEXT PRIMARY KEY, agent_id TEXT, policy_id TEXT, carrier TEXT, client TEXT, earned INT, advanced INT, paid_date DATE ); -- Enable realtime ALTER PUBLICATION supabase_realtime ADD TABLE policies; ALTER PUBLICATION supabase_realtime ADD TABLE requirements;
Your credentials are stored locally in your browser only
